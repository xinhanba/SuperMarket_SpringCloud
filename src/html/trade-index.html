Bitcoin Price Index（简称BPI），是通过采集全球范围内的主流比特币现货交易所的比特币成交价格，通过对最近1个小时的成交价格用成交量进行加权平均的形式统计出来的一个公允比特币价格指标，可以反映近一个时期的比特币价格运行状况。BTC-Q比特币期货合约将以该指数为标的，对比特币期货合约进行合约到期的交割。

交易所选择标准：在全球范围内，每3个月选择一次标的交易所。

入选需要同时满足如下要求：

  1. 日均公允比特币成交量连续3个月在5000币以上；
  2. 平台3个月内未发生重大交易事故；
  3. 平台未发现经营风险；
  4. 未出现其他可能影响该交易所稳定运行的情形；

交易所剔除的情形：

  1. 日均公允比特币成交量连续3个月在5000币以下；
  2. 平台近3个月内发生重大交易事故超过1次；
  3. 平台出现经营风险；
  4. 平台出现其他我们认为可能影响该交易所稳定运行的情形；

计价单位：统一折算成美元计价

汇率选择：以每月1日中国人民银行官网公布的汇率中间价为标准（周末和法定节假日顺延），每月矫正一次，中间不变。详见如下网址：http://www.pbc.gov.cn/publish/zhengcehuobisi/637/index.html

计算模型：采用纳入指标计算的各个平台最近1个小时价格按照实际成交量加权的加权成交平均价
